# 🧍‍♂️ Customer Segmentation using K-Means Clustering

## 📌 Project Description
This project performs customer segmentation using the K-Means clustering algorithm, an unsupervised machine learning technique that groups customers based on similarity in purchasing behavior and demographic attributes.

The application is deployed as an interactive Streamlit web application that allows users to explore customer clusters dynamically and gain insights into different customer segments.

---

## 🎯 Objective
The objective of this project is to identify distinct customer groups based on their characteristics, enabling businesses to understand customer behavior, personalize marketing strategies, and improve customer relationship management.

---

## 🧠 Machine Learning Algorithm
- K-Means Clustering
- Type: Unsupervised Learning

K-Means clustering partitions customers into K distinct clusters by minimizing the distance between data points and their respective cluster centroids.

---

## 📊 Dataset Information
The dataset includes customer-related attributes such as:

- Annual Income
- Spending Score
- Age
- Purchase Behavior
- Customer Demographics (optional features)

These features help group similar customers into meaningful clusters without using labeled outcomes.
